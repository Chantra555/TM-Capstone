# Travel Planning App

A full-stack travel planning application that allows users to manage trips, track expenses, and organize itineraries in one centralized platform. The goal of this project is to simplify travel organization through a clean UI and a secure, scalable backend.

---

## Features

- User authentication with JWT
- Create, edit, and delete trips
- Track trip expenses with real-time totals
- Separate upcoming and past trips
- User-specific data isolation (each user only sees their own data)
- Responsive and clean UI built with React

---

## Tech Stack

**Frontend**
- React
- React Router
- JavaScript (or TypeScript if used)

**Backend**
- Java
- Spring Boot
- Spring Security (JWT Authentication)
- JPA / Hibernate

**Database**
- MySQL
